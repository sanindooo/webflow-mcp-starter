The goal for this project is to publish it as a GitHub starter template.

**Why:** Enable others to clone and use the Figma-to-Webflow automation pipeline with Claude Code, Playwright visual testing, and Relume-based style guide integration.

**How to apply:** Keep the repo clean and template-friendly. Remove project-specific secrets, site IDs, and user-specific paths. Add a proper README, .gitignore, and template setup instructions. Use GitHub's "template repository" feature.
